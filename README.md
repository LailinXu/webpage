# Lailin's webpage

## Setup
1. Check out the repository
```
git clone git@github.com:LailinXu/webpage.git .
```

2. Install Hugo

```
brew install hugo
```

See also the quick-start tutorial [here](https://gohugo.io/getting-started/quick-start/).

3. Generate the webpgages for local test

```
hugo server -D
```
